# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [2.0.0] - 2025-11-03

### Adicionado
- Estrutura de diretórios profissional com `public/`, `assets/`, e `src/`
- Arquivo `portfolio.html` com projetos em destaque
- Seção institucional VPC-GEOSER no currículo
- Meta description para SEO
- Sistema de tema claro/escuro com persistência
- Links rel="noopener noreferrer" para segurança
- `.gitignore` para controle de versão
- `LICENSE` (MIT)
- Este `CHANGELOG.md`

### Alterado
- Renomeado `cv_modelo_a_v2.html` para `index.html` (facilita deploy)
- Movido PDF original para `assets/docs/` com nome sanitizado
- Atualizado README com nova estrutura de diretórios
- Expandido conteúdo de todas as seções com dados reais do PDF
- Links internos atualizados para refletir nova estrutura

### Removido
- Dados sensíveis (telefone, e-mail, CREA) conforme requisitos de privacidade
- Diretório `DOCS/` (substituído por estrutura organizada)

## [1.0.0] - 2025-11-03

### Adicionado
- Versão inicial do currículo HTML (Modelo A)
- Tema escuro/claro básico
- Layout responsivo com grid CSS
